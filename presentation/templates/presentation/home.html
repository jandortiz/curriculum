Tecnologías que manejo

En el ámbito de la programación, manejo Python y R para desarrollar soluciones eficientes y flexibles. Utilizo Jupyter Notebooks para documentar y compartir análisis interactivos, y LaTeX para la creación de documentos científicos de alta calidad.

En el campo del análisis de datos, estoy familiarizado con bibliotecas y herramientas como NumPy, Pandas, Matplotlib, y Seaborn para la manipulación y visualización de datos.

También utilizo SQL para la gestión de bases de datos y extracción de información valiosa. Además, tengo experiencia en la visualización de datos mediante herramientas como PowerBI y Tableau.

Mi conocimiento en ciencia de datos incluye habilidades en aprendizaje automático y modelos estadísticos. He trabajado con algoritmos como Regresión Lineal, K-Nearest Neighbors (KNN), Support Vector Machines (SVM), Random Forest, Redes Neuronales (NN) y Redes Neuronales Convolucionales (CNN). Utilizo herramientas como Scikit Learn, PyTorch y TensorFlow para el desarrollo y la implementación de estos modelos.

En cuanto a herramientas de desarrollo y operaciones, tengo experiencia trabajando con Git para el control de versiones, Linux como sistema operativo para desarrollo y administración de servidores, y Docker para la creación de entornos de desarrollo reproducibles y la implementación de aplicaciones en contenedores.
